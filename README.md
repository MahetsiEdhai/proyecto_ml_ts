# **Predicción de Costos de Vuelos - Proyecto MLOps**

## Descripción
Este proyecto utiliza técnicas de Machine Learning y prácticas de MLOps para predecir el costo de vuelos basado en diversos factores. Implementamos un pipeline completo desde la ingesta de datos hasta el despliegue del modelo en producción.

## Características
- Predicción de precios de vuelos basada en factores como origen, destino, aerolínea, fecha, etc.
- Pipeline de MLOps automatizado para entrenamiento, evaluación y despliegue de modelos
- Monitoreo continuo del rendimiento del modelo en producción
